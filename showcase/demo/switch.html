InputSwitch

InputSwitch is used to select a boolean value.

Basic

  • Documentation
  • Source

Getting Started

<input id="basic" type="checkbox" /></input>
                        
$('#basic').puiswitch();

Attributes

Name Type Default Description
onLabel string On Label for on state.
offLabel string Off Label for off state.

Events

Name Parameters Description
change event: puiswitchchange event
Fires when value changes.

Example

    $('input').puiswitch({
        change: function(event) {
            //..
    }});

Functions

Name Parameters Description
toggle - Slides the element.
check - Mark Checked to the checkbox.
uncheck - Mark Unchecked to the checkbox.

Example

$('#basic').puiswitch('check');

Javascript

$('#basic').puiswitch();
