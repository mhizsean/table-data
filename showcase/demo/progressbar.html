Getting Started

ProgressBar has two modes; "determinate" and "indeterminate". Former requires a value between 0 and 100 to display the progress.

$('#pb').puiprogressbar();
                        
<div id="pb" />

In order to update the value of the progress, update the value option.

$('#pb').puiprogressbar('option', 'value', 50);

Indeterminate has no such a requirement and is simple enabled using mode property.

$('#pb').puiprogressbar({indeterminate: true});

Attributes

Name Type Default Description
value number 0 Progress Bar value
labelTemplate string {value}% Progress Bar label template
showLabel boolean true Shows label on ProgressBar
effectSpeed int/string normal Defines the mode of the progress, valid values are "determinate" and "indeterminate".

Events

Name Parameters Description
complete event: puiprogressbarcomplete event Fired when value equals 100

Example

$('#default').puiprogressbar({

    }
});

Functions

Name Parameters Description

Styling

Name Element
ui-progressbar Container element.
ui-progressbar-determinate Container element of a determinate progressbar.
ui-progressbar-indeterminate Container element of an indeterminate progressbar.
ui-progressbar-value Element whose width changes according to value.
ui-progressbar-label Label element that displays the current value.

Javascript

$(function() {
    $('#pb1').puiprogressbar();

    $('#pb2').puiprogressbar({
        value: 50
    });

    $('#pb3').puiprogressbar({ 
        mode: 'indeterminate' 
    });

    Showcase.pbinterval1 = setInterval(function() {
        var val = $('#pb1').puiprogressbar('option', 'value') + 10;
        $('#pb1').puiprogressbar('option', 'value', val);
    }, 1000);
});
