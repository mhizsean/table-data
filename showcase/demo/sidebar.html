Getting Started

Sidebar is used as a container and visibility is controlled with show and hide methods.

<div id="sidebar">
    Content
</div>

$('#sidebar').puisidebar();        //initialize
$('#sidebar').puisidebar('show');  //show
$('#sidebar').puisidebar('hide');  //hide

Position

Sidebar can either be located as the left (default), right, top or bottom of the screen depending on the position property.

$('#sidebar').puisidebar({position: 'top'});

Size

Sidebar size can be changed using a fixed value or using one of the three predefined ones.

<div id="sidebar1" style="width:30em">Content</div>
<div id="sidebar2" class="ui-sidebar-sm">Content</div>
<div id="sidebar3" class="ui-sidebar-md"&>Content</div>
<div id="sidebar4" class="ui-sidebar-lg"&>Content</div>

Full Screen

Full screen mode allows the sidebar to cover whole screen.

$('#sidebar').puisidebar({fullScreen: true});

Properties

Name Type Default Description
position string left Specifies the position of the sidebar, valid values are "left" and "right".
fullScreen boolean false Adds a close icon to the header to hide the dialog.
baseZIndex number 0 Base zIndex value to use in layering.
autoZIndex boolean true Whether to automatically manage layering.

Events

Name Parameters Description
onShow - Callback to invoke when sidebar is shown.
onHide - Callback to invoke when sidebar is hidden.

Javascript

$(function() {
    $('#sb1').puisidebar({baseZIndex: 10000});
    $('#sb2').puisidebar({baseZIndex: 10000, position: 'right'});
    $('#sb3').puisidebar({baseZIndex: 10000, position: 'top'});
    $('#sb4').puisidebar({baseZIndex: 10000, position: 'bottom'});
    $('#sb5').puisidebar({baseZIndex: 10000, fullScreen: true});
    
    $('#sidebar-demo').find('button').puibutton();
    
    $('#sidebar-demo').find('button').puibutton();
});
