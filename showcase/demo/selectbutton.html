SelectButton

SelectButton is used to choose single or multiple items from a list using buttons.

Single

Multiple

  • Documentation
  • Source

Getting Started

                <div id="single" /></div>   
                        

    $('#single').puiselectbutton({
        choices: [
           {label:'Xbox One', value:0},
           {label:'PS4', value:1},
           {label:'Wii', value:2}
        ]
    });

Attributes

Name Type Default Description
choices string,number null Choices for the buttons.
formfield string null Name for the form.
tabindex string 0 Order of elements in tabbing.
multiple boolean false State for multiple button selection.

Events

Name Parameters Description
change event: puiselectbuttonchange event
Fires when value changes.

Example

    $('div').puiselectbutton({
        change: function(event) {
            //..
    }});

Functions

Name Parameters Description
selectOption value Selects the given option.
unselectOption value Unelects the given option.

Example

    $('#single').puiselectbutton('selectOption',0);

Javascript

$(function() {
    $('#single').selectbutton({
        formfield: 'single_input',
        choices: [
           {label:'Xbox One', value:0},
           {label:'PS4', value:1},
           {label:'Wii', value:2},
           {label:'PS3',value:3}
        ]
    });
    
    $('#multiple').selectbutton({
        formfield: 'multiple_input',
        choices: [
           {label:'Xbox One', value:0},
           {label:'PS4', value:1},
           {label:'Wii', value:2},
           {label:'PS3',value:3}
        ],
        multiple: true
    });
});
