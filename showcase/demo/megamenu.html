Getting Started

$('#mm').puimegamenu();
                        
<div id="mm" >
    <ul>
        <li>
            <a>
                 <div>
                    //content
                    ...
                 </div>
            </a>
        <li/>
    </ul>
</div>

Attributes

Name Type Default Description
autoDisplay boolean true Displays without the mouse click.
orientation string horizontal Defines the megamenu's orientation(vertical and horizontal).

Javascript

$('#mm').puimegamenu();

$('#vertical').puimegamenu({
    vertical:true,
    autoDisplay:false
});
