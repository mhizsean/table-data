Rating

Rating widget is a star based selection input.

Default


Default Value


Callback


No Cancel


ReadOnly

  • Documentation
  • Source

Getting Started

$('#default').puirating();
                        
<input id="rating" type="hidden" />

Attributes

Name Type Default Description
stars Integer 5 Number of stars.
cancel Boolean true Defines if user can cancel rating.

Events

Name Parameters Description
rate event: puiratingrate event
value: Rating value. 		Fired when stars selected.
oncancel event: puiratingcancel event Fired when rating is cleared.

Example

$('#default').puirating({
    rate: function(event, value) { 
        //..
    }
});

Functions

Name Parameters Description
cancel - Clears selection.
getValue - Returns current value.
setValue value: Value to update. Updates current value.
enable - Enable the widget
disable - Disable the widget.
option name: Name of the option Returns the value of the option.
option name: Name of the option, value: Value of the option Set the value of the option.

Example

$('#default').puirating('cancel');

Javascript

<script type="text/javascript">
$(function() {
    $('#rating').puirating();
        
    $('#default').puirating();
        
    $('#callback').puirating({rate: function(event, value) { 
            alert(value);
        }});
        
    $('#nocancel').puirating({cancel:false});
        
    $('#readonly').puirating({readonly:true, cancel:false});
                                 
});
</script>
