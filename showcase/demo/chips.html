Chips

Chips is used to enter multiple values on an inputfield.

Getting Started

Chips is applied on an input.

<input id="chips-basic" type="text">

$('#chips-basic').puichips();

Custom Content

A chip is customized using a template function where the value is passed as a parameter.

$('#chips-template').puichips({
    template: function(value) {
        return $('<div>' + 
                    '<span>' + value + '</span>' + 
                    '<i class="fa fa-user" style="margin-left:.5em"></i>' + 
                '</div>');
    }
});

Value

Chips component can be prepopulated using value property.

$('#chips').puichips({
    value: ['PrimeUI', 'PrimeFaces', 'PrimeNG', 'PrimeReact']
});

getValue() and setValue(value) functions are available to manage the value of the chips.

//get value
var value = $('#chips').puichips('getValue');

//set value
$('#chips').puichips('setValue', ['PrimeUI', 'PrimeFaces']);

Properties

Name Type Default Description
value array null An array as the value of the component.
name string null Name of the component to reference in form data.
field string null Name of the property to display on a chip.
disabled boolean false When present, it specifies that the element should be disabled.
style string null Inline style of the element.
styleClass string null Style class of the element.
template function null Function to provide custom content for the chip.

Events

Name Parameters Description
onAdd value: Value to add Callback to invoke when an item is added.
onRemove value: Value to remove/td> Callback to invoke when an item is removed.

Events

Name Parameters Description
getValue - Retrieves the current value of the chips.
setValue value New value to update the chips component with.
disable - Disables the component.
enable - Enables the component.
